# TAQVO Telegram Bot

Бот принимает заказы из WebApp Telegram и пересылает их вам в личку.

## Настройка

Создайте файл `.env` и добавьте в него:

BOT_TOKEN=...
OWNER_ID=...

## Запуск

npm install
npm start

Развёртывание на Render.com — подключите этот репозиторий, настройте переменные и нажмите Deploy.
